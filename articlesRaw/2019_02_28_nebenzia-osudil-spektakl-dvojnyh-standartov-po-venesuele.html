Небензя осудил "спектакль двойных стандартов по Венесуэле"

Россия и Китай наложили вето на американский проект резолюции ООН по Венесуэле, передает ТАСС. По мнению постпреда РФ при ООН Василия Небензи, документ нужен только "для дальнейших обвинений в препятствовании ввозу гуманитарной помощи”.
