Порядок распределения импортной квоты на ввоз ламп накаливания мощностью не более 200 Вт

Зарегистрирован в Минюсте РФ 29 декабря 2007 г. Регистрационный N 10842

\n

Во исполнение постановления Правительства Российской Федерации от 28 января 2006 г. N 50 "О мерах по защите российских производителей ламп накаливания" (Собрание законодательства Российской Федерации, 2006, N 6, ст.705) приказываю:

\n

1. Утвердить прилагаемый Порядок распределения импортной квоты на ввоз на таможенную территорию Российской Федерации в таможенном режиме выпуска для внутреннего потребления электрических ламп накаливания мощностью не более 200 Вт и на напряжение более 100 В, за исключением ламп ультрафиолетового или инфракрасного излучения, галогенных ламп с вольфрамовой нитью и рефлекторных ламп, классифицируемых кодом ТН ВЭД России 8539 22 900 0 (далее - лампы накаливания), между участниками внешнеэкономической деятельности.

\n

2. Департаменту государственного регулирования внешнеторговой деятельности (Г.В. Баландина) ежегодно:

\n

а) на основании данных таможенной статистики, представленных ФТС России, распределять импортную квоту на ввоз ламп накаливания для участников внешнеэкономической деятельности (далее - участники ВЭД) с разбивкой по государствам-экспортерам (союзу государств) в соответствии с Порядком, указанным в пункте 1 настоящего приказа;

\n

б) разместить на веб-сайте Минэкономразвития России перечни участников ВЭД с указанием распределения им импортной квоты на ввоз ламп накаливания.

\n

3. Территориальным органам Минэкономразвития России осуществлять в установленном порядке выдачу разовых лицензий после получения перечней участников ВЭД с распределенными им импортными квотами.

\n

4. Признать утратившим силу приказ Министерства экономического развития и торговли Российской Федерации от 19 декабря 2006 г. N 415 "О Порядке распределения импортной квоты на ввоз на таможенную территорию Российской Федерации в таможенном режиме выпуска для внутреннего потребления электрических ламп накаливания мощностью не более 200 Вт и на напряжение более 100 В, за исключением ламп ультрафиолетового или инфракрасного излучения, галогенных ламп с вольфрамовой нитью и рефлекторных ламп, классифицируемых кодом ТН ВЭД России 8539 22 900 0" (зарегистрирован в Минюсте России 17 января 2007 г., регистрационный номер 8771, Бюллетень нормативных актов федеральных органов исполнительной власти, N 6).

\n

Порядок распределения импортной квоты на ввоз на таможенную территорию Российской Федерации в таможенном режиме выпуска для внутреннего потребления электрических ламп накаливания мощностью не более 200 Вт и на напряжение более 100 В, за исключением ламп ультрафиолетового или инфракрасного излучения, галогенных ламп с вольфрамовой нитью и рефлекторных ламп, классифицируемых кодом ТН ВЭД России 8539 22 900 0

\n

1. Исходные данные, используемые при распределении квоты. При распределении квоты использованы следующие показатели:

\n

- суммарные (по участникам ВЭД) объемы фактического ввоза ламп накаливания за период с 1 октября предыдущего года по 30 сентября текущего года в разрезе стран ввоза;

\n

- объемы выделенных квот на ввоз ламп накаливания в разрезе стран ввоза.

\n

2. Базовым принципом является сохранение доли каждого участника ВЭД в общем объеме завоза ламп накаливания за период с 1 октября предыдущего года по 30 сентября текущего года при расчете квот на следующий календарный год.

\n

3. Квоты по странам на ввоз ламп накаливания для участника ВЭД рассчитываются по следующим формулам:

\n

Кобщ = Vкв.общ / V общ;

\n

Кевр = Vкв.евр / Vевр;

\n

Ккир = Vкв.кир / Vкир;

\n

Ккит = Vкв.общ * Vкит / Vобщ;

\n

Кукр = Vкв.укр / Vукр;

\n

Кпроч = Vкв.общ * Vпроч / Vобщ;

\n

Vкв.евр = Vевр * Кевр;

\n

Vкв.кир = Vкир * Ккир;

\n

Vкв.кит = Кобщ * Vкит + ((Vкв.кит - Ккит) / (Vкв.кит + Vкв.проч - Ккит) *

\n

(Кобщ * (Vевр + Vкир + Vукр)) - Кевр - Ккир - Кукр);

\n

Vкв.укр = Vукр * Кукр;

\n

Vкв.проч = Кобщ * Vпроч + ((Vкв.проч) / (Vкв.проч + Vкв.кит - Кпроч) * (Кобщ * (Vевр + Vкир + Vукр)) - Кевр - Ккир - Кукр).

\n

4. Общая квота на ввоз ламп накаливания для участника ВЭД рассчитывается по следующей формуле:

\n

Vкв.общ = Vкв.евр + Vкв.кир + Vкв.кит + Vкв.укр + Vкв.проч.

\n

Обозначения, используемые в формулах

\n

Кобщ - общий коэффициент изменения объема ввоза всех участников ВЭД по квоте следующего года по сравнению с 1 октября предыдущего года по 30 сентября текущего года;

\n

Vевр - объем фактического ввоза ламп накаливания участником ВЭД из стран Европы с 1 октября предыдущего года по 30 сентября текущего года;

\n

Vкир - объем фактического ввоза ламп накаливания участником ВЭД из Киргизии с 1 октября предыдущего года по 30 сентября текущего года;

\n

Vкит - объем фактического ввоза ламп накаливания участником ВЭД из Китая с 1 октября предыдущего года по 30 сентября текущего года;

\n

Vукр - объем фактического ввоза ламп накаливания участником ВЭД из Украины с 1 октября предыдущего года по 30 сентября текущего года;

\n

Vпроч - объем фактического ввоза ламп накаливания участником ВЭД из прочих стран (т.е. всех стран, за исключением стран Европы, Киргизии, Китая, Украины) с 1 октября предыдущего года по 30 сентября и текущего года;

\n

Vкв.евр - объем выделенной квоты ввоза ламп накаливания для участника ВЭД из стран Европы на следующий год;

\n

Vкв.кир - объем выделенной квоты ввоза ламп накаливания для участника ВЭД из Киргизии на следующий год;

\n

Vкв.кит - объем выделенной квоты ввоза ламп накаливания для участника ВЭД из Китая на следующий год;

\n

Vкв.укр - объем выделенной квоты ввоза ламп накаливания для участника ВЭД из Украины на следующий год;

\n

Vкв.проч - объем выделенной квоты ввоза ламп накаливания для участника ВЭД из прочих стран (т.е. всех стран, за исключением стран Европы, Киргизии, Китая и Украины) на следующий год;

\n

Vкв.общ. - общий объем выделенной квоты ввоза ламп накаливания для участника ВЭД на следующий год;

\n

К - коэффициент изменения суммарного объема ввоза по квоте следующего года по сравнению с ввозом с 1 октября предыдущего года по 30 сентября текущего года;

\n

Кевр - коэффициент изменения объема ввоза из стран ЕС по квоте следующего года по сравнению с 1 октября предыдущего года по 30 сентября текущего года;

\n

Ккир - коэффициент изменения объема ввоза из Киргизии по квоте следующего года по сравнению с 1 октября предыдущего года по 30 сентября текущего года.

\n

Ккит - коэффициент изменения объема ввоза из Китая по квоте следующего года по сравнению с 1 октября предыдущего года по 30 сентября текущего года.

\n

Кукр - коэффициент изменения объема ввоза из Украины по квоте следующего года по сравнению с 1 октября предыдущего года по 30 сентября текущего года.

\n

Кпроч - коэффициент изменения объема ввоза из прочих стран по квоте следующего года по сравнению с 1 октября предыдущего года по 30 сентября текущего года.
